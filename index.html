README

Welcome to the repository of PAXelerate - an open source passenger flow simulation framework for advanced aircraft cabin layouts!

  • The current version under development is Alpha 0.6
  • The implementation began in November 2014

Quick Guide

  1. Install Eclipse Mars - Modelling Tools and Java JDK Version 8
  2. Upgrade the EMF Client Platform to version 1.7.x
  3. Checkout OpenCDT core plugins
  4. Checkout PAXelerate
  5. Launch paxelerate.product

How do I get set up?

  • The Installation Guide is intended to enable a quick and easy setup
  • The Tutorial provides a summary of the functional range and quick introduction for new users
  • The Developers Guide summarises useful tips to get started with the development
